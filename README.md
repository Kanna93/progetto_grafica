# PROGETTO AUTOTRUCK

## Progetto del corso di Grafica della Laurea Magistrale in Informatica

Il progetto è stato sviluppato in linguaggio C++ per ambiente Windows utilizzando le librerie OpenGL/SDL2.

Tutte le informazioni dettagliate inerenti al progetto sono consultabili al sito https://mattiabenatti.github.io/progetto_grafica/

Per installare ed eseguire il videogioco, è necessario avviare il terminale di MinGW all'interno della cartella scaricata ed utilizzare il comando `make run`

I comandi principali sono i seguenti:
- **W**: Accelera
- **S**: Frena/Retromarcia
- **A**: Curva a Sinistra
- **D**: Curva a Destra
- **F1**: Cambia Camera
- **F2**: Attiva/Disattiva Wireframe
- **F3**: Accendi/Spegni fari
- **F4**: Attiva/Disattiva Ombre
- **O**: Menu Principale/Pausa
- **SHIFT+NUM**: Modalità Notte
- **Q**: Esci
- **MOUSE**: Muovi Camera Fissa

La funzione demo commentata nel file *car.cpp* è ancora in fase di sviluppo e prevederà un giro di dimostrazione dove l'auto guida in maniera autonoma.
